Find the basis for a decision

Submitted by admin on Wed, 05/09/2018 - 00:54
Contributor: 
Mikael Borg
RDA Group: 
Storage Service Definitions WG
Goal: 
To find the basis for a decision
Summary: 
The actor need to have a full trace of the documentation underlying a decision for e.g. access to a data set. For example: <ul> <li> access policies </li> <li> data access application</li> <li> data access agreement</li> </ul>

Actors:

  • Data Access Committee
  • Data Owner
  • Data User
  • System Maintainer
Broader UC: 
Trace the agent who is responsible for a decision
